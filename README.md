# BeyondChats Assignment

## Tech Stack
- Node.js
- Express
- React (Vite)
- SQLite

## Project Structure
- beyondchats-scraper/
- beyondchats-backend/
- beyondchats-ui/

## Local Setup

### Backend
cd beyondchats-backend
npm install
node server.js

### Frontend
cd beyondchats-ui
npm install
npm run dev

## Data Flow
Scraper → SQLite → API → React UI
